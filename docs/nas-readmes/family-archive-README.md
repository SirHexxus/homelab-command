# Family Archive

Long-term personal archive for the Chang family — David and Lorna's files, wedding records,
and important household documents.

---

## File Structure

```text
family-archive/
├── chang-files/          # David and Lorna's personal files
├── my-wedding/           # Wedding photos, documents, and records
└── important-files/      # Wills, official documents, and critical household records
```

---

## Naming Schema

- **Folders:** fixed — do not add subdirectories without intention
- **Files:** preserve original filenames; no rename convention enforced

---

## Consumers

| Application | Platform | Role |
|---|---|---|
| Direct file access | NFS / any client | Manual retrieval only — not served by any application |

---

## Notes

- This share is for long-term preservation, not active use
- No automated management — manual additions only
- `important-files/` contains sensitive documents (wills, legal records) — access accordingly
